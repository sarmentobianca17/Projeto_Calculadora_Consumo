### ➕ Calculadora de consumo mensal de energia

<div align="center">
  <img alt="Pinguim utilizando uma calculadora" src="https://media.tenor.com/2k4vKSclZSMAAAAj/calculator-cute.gif"/>
</div>
<div>
<ul>
<li>
<b>
Nome e objetivo do sistema:
</b> possibilitar ao usuário o cálculo de consumo mensal em R$/kWh de um eletrodoméstico
</li>
<li>
<b>
Fórmula utilizada para o cálculo de consumo mensal: 
</b> (potencia * horasDia * 30) / 1000
</li>
<li>
<b>
Fórmula utilizada para o cálculo do custo estimado em R$/kWh: 
</b> (consumo) * Valor fixo em R$/kWh 
</li>
<li>
<b>
Instruções para realizar o cálculo: 
</b>
<ol>
1. Digite o nome do eletrodoméstico
</ol>
<ol>
2. Digite a potência do aparelho em watts (W)
</ol>
<ol>
3. Digite o tempo médio de uso diário em horas
</ol>
</li>
